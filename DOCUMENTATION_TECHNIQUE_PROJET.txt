DOCUMENTATION TECHNIQUE DU PROJET
=================================

Projet : Couture Document API
Objectif : API REST moderne et evolutive pour gerer un catalogue documentaire specialise couture.

REGLE DE MAINTENANCE OBLIGATOIRE
--------------------------------

Ce fichier doit etre mis a jour a chaque modification du projet.

A chaque ajout, modification ou suppression de code, documenter ici :
- ce qui a change ;
- dans quel fichier ;
- quelle technologie est concernee ;
- a quoi sert la modification ;
- si la modification impacte les commandes, la configuration, la base de donnees, les routes ou les tests.


STACK TECHNIQUE
---------------

Python 3.12+
- Langage backend principal.

FastAPI
- Framework HTTP.
- Expose les routes REST, Swagger /docs, ReDoc /redoc et l'injection de dependances.

PostgreSQL
- Base relationnelle principale.
- Utilisee avec UUID, enums et listes PostgreSQL.

SQLAlchemy 2.x async
- ORM moderne.
- Utilise AsyncSession, mapped_column, Mapped et create_async_engine.

Pydantic v2
- Validation des entrees/sorties API.
- Utilise BaseModel, field_validator et model_validator.

pydantic-settings
- Configuration par variables d'environnement.

Alembic
- Migrations de schema de base de donnees.

Docker / Docker Compose
- Lance l'API et PostgreSQL.

pytest / httpx
- Tests automatises async de l'API.

Ruff
- Controle qualite et imports.

WORKFLOW DOCUMENTAIRE
---------------------

draft
- Brouillon.
- Peut etre incomplet.
- Sert aux benevoles qui commencent une fiche.

pending_validation
- Fiche prete pour relecture.
- Requiert isbn, title et authors.

validated
- Fiche validee.
- Requiert isbn, title et authors.
- validated_at est renseigne automatiquement si absent.

COMMANDES UTILES
----------------

Lancer le projet :
docker compose up --build

Appliquer les migrations :
docker compose exec api alembic upgrade head

Creer une migration :
docker compose exec api alembic revision --autogenerate -m "description"

Lancer les tests :
pytest

Controle qualite :
ruff check .

Lancer le frontend :
cd frontend
npm install
npm run dev

Verifier le frontend :
cd frontend
npm run typecheck
npm run build
